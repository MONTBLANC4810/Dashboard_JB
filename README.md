# 📊 전북지역본부 경영현황 대시보드 (2021-2026)

이 프로젝트는 전북지역본부의 연도별 매출 및 경영 지표를 시각화하여 분석하기 위한 전문 비즈니스 인텔리전스(BI) 대시보드입니다.

## 🚀 주요 기능
- **엑셀 데이터 자동 파싱**: `2021-2026.2월 개별매출현황.xlsx` 파일을 드래그 앤 드롭하여 즉시 시각화.
- **가변 KPI 게이지**: 100% 달성 시 자동으로 120% 스케일로 확장되는 지능형 게이지 UI.
- **다차원 필터링**: 연도, 월, 부서, 고객사, 예산 항목별 멀티 셀렉션 필터 지원.
- **시계열 분석**: 연도별 월 실적 비교 및 상위 고객 매출 추이 분석.
- **반응형 레이아웃**: 모든 모니터 해상도에서 한 화면에 최적화된 Grid 시스템.

## 🛠 기술 스택
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS v4 (Glassmorphism & Modern UI)
- **Charts**: Recharts
- **Data**: SheetJS (xlsx)

## 📦 시작하기
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
